This repository is a **version archive** for the  
**ESP32 Universal CC/CV Charger Controller** project.

It stores **ZIP and RAR files** of all major firmware releases so that every
working version is preserved and easily accessible.

This repository is **not for active development**.

---

## Why This Repository Exists

- Preserve all stable and historical firmware versions  
- Allow rollback to known working releases  
- Maintain a permanent record of design evolution  
- Enable safe experimentation without breaking main development  
- Provide offline access to complete, build-ready snapshots
